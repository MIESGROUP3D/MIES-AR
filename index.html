Cómo usarlo

  1. 1Visualiza: arrastra con el dedo o el ratón para rotar el modelo.
  2. 2Inicia AR: pulsa “Ver en tu espacio” y se activará la cámara.
  3. 3Detecta: mueve el dispositivo hasta encontrar una superficie plana.
  4. 4Coloca: el modelo se ancla automáticamente a la superficie.
  5. 5Interactúa: con dos dedos mueve, escala o rota el objeto.
Nota técnica: usa WebXR. Requiere Android con ARCore o iPhone/iPad con ARKit (Safari).

Soporte y dispositivos

iOS (iPhone/iPad): requiere iOS 12+. Safari abre automáticamente el visor AR de ARKit (Quick Look).

Android: requiere Chrome y un dispositivo compatible con ARCore (servicios de Google Play).

PC de escritorio: la función AR no está disponible, pero puedes usar la vista previa 3D interactiva.

Si el modelo no carga, verifica que el archivo .glb no esté dañado.
